# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text — including standards, enforcement responsibilities, scope,
enforcement guidelines, and attribution — is published at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report a concern, contact the maintainer at **marc.hyett@gmail.com**.
Reports are kept confidential. The maintainer will acknowledge receipt within
five business days and follow up with next steps.
